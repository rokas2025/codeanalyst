Connect your WordPress site to CodeAnalyst for advanced theme analysis and code insights.

== Description ==

CodeAnalyst Connector is a powerful plugin that connects your WordPress website to the CodeAnalyst platform, enabling:

* Theme file analysis (PHP, CSS, JavaScript)
* Content readability analysis
* Site health monitoring  
* WordPress version tracking
* Plugin and theme compatibility checks

== Features ==

* **Secure Connection**: Uses unique API keys for secure communication
* **Theme File Reading**: Analyzes your active theme files for best practices
* **Content Analysis**: Evaluates post and page content
* **Daily Sync**: Automatically syncs site data every 24 hours
* **Real-time Testing**: Test your connection anytime

== Installation ==

1. Upload the `codeanalyst-connector` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to CodeAnalyst settings page
4. Get your API key from [CodeAnalyst Settings](https://codeanalyst.vercel.app/settings)
5. Enter the API key and click "Connect to CodeAnalyst"

== Frequently Asked Questions ==

= Where do I get an API key? =

Sign in to your CodeAnalyst account at https://codeanalyst.vercel.app, go to Settings, and generate a WordPress API key.

= Is my data secure? =

Yes! All communication is encrypted and your API key is stored securely in your WordPress database.

= What files are analyzed? =

The plugin analyzes PHP, CSS, and JavaScript files from your active theme. It does not modify any files.

= Can I disconnect anytime? =

Yes, you can disconnect from CodeAnalyst at any time from the plugin settings page.
