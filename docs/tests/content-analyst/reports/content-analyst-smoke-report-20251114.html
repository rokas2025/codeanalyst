What is a Smoke Test?

Smoke Test Definition

Smoke testing is a preliminary performance test that validates basic system functionality and health after deployment. Think of it as a "sanity check" before running more intensive tests.

Characteristics:

  • Low load (0.2-0.5 requests/second)
  • Short duration (5-10 minutes)
  • Validates critical paths work correctly
  • Quick feedback on deployment issues
  • Baseline for comparison with other test types
