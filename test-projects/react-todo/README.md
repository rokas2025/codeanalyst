# React Todo App

A modern, well-tested Todo application built with React and TypeScript.

## Features

- ✅ Add, edit, and delete todos
- ✅ Mark todos as complete
- ✅ Filter by status
- ✅ Persistent storage
- ✅ 90%+ test coverage
- ✅ Full TypeScript support

## Installation

```bash
npm install
npm start
```

## Testing

```bash
npm test
```

## Architecture

- **Components**: Reusable UI components
- **Hooks**: Custom React hooks for state management
- **Types**: Full TypeScript definitions
- **Tests**: Comprehensive unit and integration tests

## Code Quality

- ESLint configured
- Prettier formatting
- Jest for testing
- 90%+ test coverage
